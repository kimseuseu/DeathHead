# DeathHead

R.E.P.O. 스타일 사망 연출 + 아이템 봉인/회수 시스템 Paper 플러그인

## 기능

- **사망 연출** — 항공샷 카메라, 폭죽, 파티클, 타이틀
- **아이템 봉인** — 사망 시 인벤토리의 10~15%를 플레이어 머리에 봉인하여 드롭
- **머리 회수** — 머리를 설치 후 열쇠로 우클릭하면 봉인 해제
- **부패 시스템** — 일정 시간(기본 60분) 후 머리가 부패하여 아이템 영구 유실
- **TextDisplay** — 드롭된 머리 위에 소유자 이름 + 남은 시간 표시
- **발광 효과** — 드롭된 머리가 블록 뒤에서도 보임
- **사망 방지권** — 소지 시 사망해도 아이템 유실 없음 (1개 소모)
- **미리보기** — 인벤토리에서 머리를 Shift+우클릭하면 봉인된 아이템 확인 가능
- **크래프팅** — 열쇠 제작 레시피 (철 + 금 + 막대기)

## 명령어

| 명령어 | 설명 | 권한 |
|--------|------|------|
| `/dh` | 도움말 | - |
| `/dh key [플레이어] [개수]` | 열쇠 지급 | `deathhead.admin` |
| `/dh ticket [플레이어] [개수]` | 사망 방지권 지급 | `deathhead.admin` |
| `/dh reload` | 설정 리로드 | `deathhead.admin` |

## 요구사항

- Paper 1.21.11+
- Java 21+

## 빌드

```bash
mvn clean package
```

`target/DeathHead-2.1.0.jar` 생성 → `plugins/` 폴더에 배치

## 설정

`config.yml`에서 패널티 비율, 부패 시간, 열쇠 재료, 메시지 등을 커스텀할 수 있습니다.
